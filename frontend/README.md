# Frontend - React.js Application

This directory contains the React.js frontend application for the Esoteric loan management platform.

## Features

- Modern purple/black UI design
- Loan dashboard with growth charts
- Two-factor authentication
- Responsive design for mobile and desktop

## Technology Stack

- React.js
- Material-UI
- Chart.js
- Axios
- React Router

## Getting Started

```bash
npm install
npm start
```
